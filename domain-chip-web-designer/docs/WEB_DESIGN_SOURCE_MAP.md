## Award and Inspiration Sources

- `Awwwards` - reference for jury-shaped website craft, interaction ambition, and current showcase patterns
- `The FWA` - reference for digital experience experimentation and interactive campaign caliber
- `CSS Design Awards` - reference for judged web craft across UX, UI, and innovation
- `SiteInspire` - reference for pattern clustering by industry, type, and style tags
- `Godly` - reference for strong modern website inspiration and interaction packaging
- `Land-book` - reference for landing-page pattern comparison and conversion-surface scouting

## Durable UI/UX Sources

- `W3C WCAG 2.2` - accessibility guardrail baseline
- `Nielsen Norman Group` - usability heuristics and interaction clarity reference
- `Apple Human Interface Guidelines` - system-level interaction and interface discipline
- `Material Design` - component and product-UX reference for structured app surfaces

## Use Policy

Use award sources to learn:

- composition
- pacing
- motion restraint
- typography systems
- visual hierarchy
- portfolio and storytelling patterns

Use UX sources to validate:

- task flow
- semantic structure
- accessibility
- interaction predictability
- decision clarity

Do not treat inspiration galleries as proof of product usability.

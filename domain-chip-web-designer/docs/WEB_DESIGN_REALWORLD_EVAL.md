# Web Design Real-World Eval

This chip separates benchmark beauty from live product truth.

## Landing Pages

Track:

- first-screen comprehension
- CTA click-through or demo-start rate
- form completion quality
- bounce after hero
- proof section engagement
- speed and performance budgets

## Dashboards

Track:

- task completion rate
- time to first successful action
- navigation error rate
- confusion during handoff or setup
- repeat-task speed after first use
- keyboard and screen-reader viability

## Portfolio or Studio Sites

Track:

- whether work is understandable without narration
- whether case studies explain decisions, not just outcomes
- whether motion helps reveal craft instead of hiding weak content
- whether contact or inquiry paths are obvious

## Hard Rule

If the live surface shows confusion, hesitation, or accessibility breakage, do not let benchmark style scores overrule it.

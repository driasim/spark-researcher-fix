# Agentic Marketing Real-World Eval

Use this once a benchmark candidate is ready for live rollout.

## Primary Questions

1. Did the system create qualified demand, not just impressions?
2. Did it shorten the path from new signal to human decision?
3. Did it improve the startup's understanding of buyer language and objections?
4. Can the team run it without hidden heroics?

## Real-World Metrics

Track at minimum:

- attributable leads created
- qualified pipeline created
- time from signal to first routed touch
- time from touch to human review
- response rate or meeting rate for outbound motions
- conversion rate for inbound motions
- survey completion or conversation feedback rate
- content-to-pipeline lag

## Real-World Promotion Rule

Only promote benchmark doctrine to real-world validated doctrine when:

- the live pilot created attributable qualified pipeline
- the feedback loop produced a clear message improvement
- the workload stayed within the startup's actual operating capacity
- the result can be repeated without changing the whole system each week

## Root Lesson

Approved self-edits should support review-branch promotion, not only manual apply.

## Lineage Failures

- Earlier self-edit flow stopped at proposal packets.
- There was no branch workflow for human review before merge.
- Direct-main and review-branch preferences were not configurable per owner.

## Counterfactual

Without review-branch promotion, approved self-edits still require manual git operations outside the system.
